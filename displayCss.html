Layout CSS - A propriedade display

A propriedade display é a propriedade CSS mais importante para controlar o layout.

A propriedade de display

A propriedade display especifica se/como um elemento é exibido.

Cada elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento. O valor de exibição padrão para a maioria dos elementos é block ou inline.

Elementos de nível de bloco

Um elemento de nível de bloco sempre começa em uma nova linha e ocupa toda a largura disponível (estende-se para a esquerda e para a direita o máximo possível).

O elemento <div> é um elemento de nível de bloco.

Exemplos de elementos de nível de bloco:

  • <div>>
  • <h1> - <h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>

Elementos embutidos

Um elemento inline não começa em uma nova linha e ocupa apenas a largura necessária.

Este é um elemento <span> embutido dentro de um parágrafo.

Exemplos de elementos embutidos:

  • <span>>
  • <ul>
  • <img>

Display none

display: none; é comumente usado com JavaScript para ocultar e mostrar elementos sem excluí-los e recriá-los.

O elemento <script> usa display: none; como padrão.

Exemplo

   <head>
   <style>
   li.lista {
      display: inline;
   }
   </style>
   </head>

   <body>
   <p>Display a list of links as a horizontal menu:</p>
   
   <ul>
      <li class="lista"><a href="//www.w3schools.com/html/default.asp" 
         target="_blank">HTML</a></li>
      <li class="lista"><a href="//www.w3schools.com/css/default.asp" 
         target="_blank">CSS</a></li>
      <li class="lista"><a href="//www.w3schools.com/js/default.asp" 
         target="_blank">JavaScript</a></li>
   </ul>
   </body>

Observação: Definir a propriedade de exibição de um elemento altera apenas como o elemento é exibido, NÃO o tipo de elemento. Portanto, um elemento inline com display: block;não pode ter outros elementos de bloco dentro dele.

Substituir o valor de exibição padrão

Conforme mencionado, cada elemento tem um valor de exibição padrão. No entanto, você pode substituir isso.

Alterar um elemento inline para um elemento block, ou vice-versa, pode ser útil para fazer a página parecer de uma maneira específica e ainda seguir os padrões da web.

Ocultar um elemento - exibição: nenhum ou visibilidade: oculto?

Ocultar um elemento pode ser feito definindo a propriedade display como none. O elemento ficará oculto e a página será exibida como se o elemento não estivesse lá:

      <head>
      <style>
         h3.hidden {
            display: none;
         }
      </style>
      </head>

      <body>
         <h2>Este é um título visível</h2>
         <h3 class="hidden">Este é um título oculto</h3>
         <p>Observe que o elemento <h3> com display: none; não ocupa espaço.</p>
      </body>

Resultado do Exemplo Acima

Este é um título visível

Este é um título oculto

Observe que o elemento h3 com display: none; não ocupa espaço.

visibility:hidden;também esconde um elemento.

No entanto, o elemento ainda ocupará o mesmo espaço de antes. O elemento ficará oculto, mas ainda afetará o layout:

      <head>
      <style>
         h3.hidden2 {
            visibility: hidden;
         }
      </style>
      </head>

      <body>
         <h2>Este é um título visível</h2>
         <h3 class="hidden2">Este é um título oculto</h3>
         <p>Observe que o elemento <h3> com visibility: hidden; ocupa o mesmo espaço.</p>
      </body>

Resultado do Exemplo Acima

Este é um título visível

Este é um título oculto

Observe que o elemento h3 com visibility: hidden; ocupa o mesmo espaço .
