Margens CSS

As propriedades margin CSS são usadas para criar espaço ao redor dos elementos, fora de qualquer borda definida.

Com CSS, você tem controle total sobre as margens. Existem propriedades para definir a margem para cada lado de um elemento (superior, direita, inferior e esquerda).

Margem - Lados Individuais

CSS tem propriedades para especificar a margem para cada lado de um elemento:

  • margin - propriedade abreviada define todas as propriedades
  • margin-top - margem superior
  • margin-right - margem direita
  • margin-bottom - margem inferior
  • margin-left - margem esquerda

Todas as propriedades de margem podem ter os seguintes valores:

  • auto - o navegador calcula a margem
  • comprimento - especifica uma margem em px, pt, cm, etc.
  • % - especifica uma margem em % da largura do elemento recipiente
  • inherit - especifica que a margem deve ser herdada do elemento pai

Dica: Valores negativos são permitidos.

Exemplo

Defina margens diferentes para todos os quatro lados de um elemento <p>:

      div {
         background-color: lightblue;
         border: 1px solid black;
         margin-top: 100px; /* margem superior 100px */
         margin-bottom: 100px;/* margem inferior 100px */
         margin-right: 150px; /* margem direita com 150px */
         margin-left: 80px; /* margem esquerda com 80px */
      }

Proriedade abreviada margin

Para encurtar o código, é possível especificar todas as propriedades de margem em uma propriedade abreviada margin.

A propriedade margin é uma propriedade abreviada para as seguintes propriedades de margem individuais:

      Exemplo de margem individual.
      margin-top     // margem superior
      margin-right   // margem direita
      margin-bottom  // margem inferior
      margin-left    // margem esquerda

Se a propriedade margin tiver quatro valores:

  • margem: 25px 50px 75px 100px;
    • margem superior é 25px
    • margem direita é 50px
    • margem inferior é 75px
    • margem esquerda é 100px

Use a propriedade abreviada margin com quatro valores:

      Exemplo
      p {
        /* propriedade abreviada */ 
        margin: 25px 50px 75px 100px;
      }

Se a propriedade margin tiver três valores:

  • margem: 25px 50px 75px;
    • margem superior é 25px
    • as margens direita e esquerda são 50px
    • margem inferior é 75px

Use a propriedade abreviada de margem com três valores: 

      p {
        // propriedade abreviada com três valores
        margin: 25px 50px 75px;
      }

Se a propriedade margin tiver dois valores:

  • margem: 25px 50px;
    • as margens superior e inferior são 25px
    • as margens direita e esquerda são 50px

Use a propriedade abreviada de margem com dois valores:

      propriedade abreviada com dois valores
      p {
        margin: 25px 50px;
      }

Se a propriedade margin tiver um valor:

  • margem: 25px;
    • todas as quatro margens são 25px

Use a propriedade abreviada de margem com um valor:

      Propriedade abreviada com um valor.
      p {
        margin: 25px;
      }

O valor automático

Você pode definir a propriedade margin: auto para centralizar horizontalmente o elemento em seu contêiner.

O elemento ocupará a largura especificada e o espaço restante será dividido igualmente entre as margens esquerda e direita.

      Exemplo: Usando margem: auto:
      div {
         width: 300px;
         margin: auto;
         border: 10px solid red;
      }
