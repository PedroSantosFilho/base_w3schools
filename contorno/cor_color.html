Cor do Contorno CSS

A propriedade outline-color é usada para definir a cor do contorno.

A cor pode ser definida por:

  • nome - especifique um nome de cor, como "vermelho"
  • HEX - especifique um valor hexadecimal, como "#ff0000"
  • RGB - especifique um valor RGB, como "rgb(255,0,0)"
  • HSL - especifique um valor HSL, como "hsl(0, 100%, 50%)"
  • inverter - realiza uma inversão de cores (garante que o contorno fique visível, independentemente da cor do fundo)

O exemplo a seguir mostra alguns contornos diferentes com cores diferentes. Observe também que esses elementos também possuem uma borda branca fina dentro do contorno:

      <style>
         p.ex1 {
            border: 5px solid white;
            outline-width: 5px;
            outline-style: solid;
            outline-color: red;
         }

         p.ex2 {
            border: 5px solid white;
            outline-width: 5px;
            outline-style: dotted;
            outline-color: blue;
         }

         p.ex3 {
            border: 5px solid white;
            outline-width: 5px;
            outline-style: outset;
            outline-color: grey;
         }
      </style>
