Largura do Contorno CSS

A propriedade outline-width especifica a largura do contorno e pode ter um dos seguintes valores:

  • thin ( fino - normalmente 1px)
  • medium ( médio - normalmente 3px)
  • thick ( grosso - normalmente 5px)
  • Um tamanho específico (em px, pt, cm, em, etc)

A propriedade outline

O exemplo a seguir mostra alguns contornos com diferentes larguras:

   <style
      p#exp1 {
         border: 1px solid white; => borda: largura 1px, 
                                             tipo sólido 
                                             cor branca
         outline-style: solid; => estilo do contorno: sólido
         outline-color: red; => cor do contorno: vermelho
         outline-width: thin; => largura do contorno: fino
         padding: 5px; => enchimento: 5px
         margin: 20px 200px 5px 100px; => margem: topo 20px,
                                             lado direito 200px,
                                             inferior 5px,
                                             lado esquerdo 100px
      }

      p#exp2 {
         border: 1px solid white;
         outline-style: solid;
         outline-color: red;
         outline-width: medium; => largura do contorno: médio
         margin: 20px 200px 2px 100px;
         padding: 5px;
      }

      p#exp3 {
         border: 1px solid white;
         outline-style: solid;
         outline-color: red;
         outline-width: thick; => largura do contorno: grosso
         margin: 20px 200px 2px 100px;
         padding: 5px;
      }

      p#exp4 {
         border: 1px solid white;
         outline-style: solid;
         outline-color: red;
         outline-width: 4px; => largura do contorno: 4 pixel
         margin: 20px 200px 2px 100px;
         padding: 5px;
      }
   </style>
