Cor da Borda do CSS

A propriedade border-color é usada para definir a cor das quatro bordas.

A cor pode ser definida por:

  • name - especifique um nome de cor, como "red" (vermelho)
  • HEX - especifique um valor HEX, como "#ff0000"
  • RGB - especifique um valor RGB, como "rgb(255,0,0)"
  • HSL - especifique um valor HSL, como "hsl(0, 100%, 50%)"
  • transparent

Nota: Se border-color não estiver definido, herda a cor do elemento.

   Exemplo
   p.one {
      border-style: solid;
      border-color: red; /* vermelho */
      border-width: 10px; /* largura 10 px */
      padding: 10px; /* espaços entre o texto e as bordas */
    }
    
    p.two {
      border-style: solid;
      border-color: green; /* verde */
      border-width: 10px; /* largura 10 px */
      padding: 10px; /* espaços entre o texto e as bordas */
    } 
    
    p.three {
      border-style: dotted;
      border-color: blue; /* azul */
      border-width: 10px; /* largura 10 px */
      padding: 10px; /* espaços entre o texto e as bordas */
    }

Cores Laterais Específicas

A propriedade border-color: pode ter de um a quatro valores (para a borda superior, borda direita, borda inferior e borda esquerda).

   Exemplo
   p.um {
      border-style: solid; /* estilo sólido */
      
      border-width: 10px; /* largura 10 px */
      
      border-color: red green blue yellow; 
      /* cor topo vermelho, direito verde, 
      fundo azul e esquerdo amarelo */
      
      padding: 10px; /* espaços entre o texto e as bordas */
   }
