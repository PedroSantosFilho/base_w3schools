Preenchimento CSS

O preenchimento é usado para criar espaço ao redor do conteúdo de um elemento, dentro de qualquer borda definida.

Com CSS, você tem controle total sobre o preenchimento. Existem propriedades para definir o preenchimento de cada lado de um elemento (superior, direito, inferior e esquerdo).

Preenchimento - Lados Individuais

CSS tem propriedades para especificar o preenchimento de cada lado de um elemento:

  • padding - propriedade abreviada define todas as propriedades
  • padding-top - preenchimento superior
  • padding-right - preenchimento direita
  • padding-bottom - preenchimento inferior
  • padding-left - preenchimento esquerda

Todas as propriedades de preenchimento podem ter os seguintes valores:

  • length - especifica um preenchimento em px, pt, cm, etc.
  • % - especifica um preenchimento em % da largura do elemento que o contém
  • inherit - especifica que o preenchimento deve ser herdado do elemento pai

Dica: Valores negativos são permitidos.

Exemplo

Define um preenchimento diferentes para todos os quatro lados de um elemento <div>:

      div {
         background-color: lightblue;
         border: 1px solid black;
         padding-top: 100px; /* preenchimento superior 100px */
         padding-bottom: 100px;/* preenchimento inferior 100px */
         padding-right: 150px; /* preenchimento direita com 150px */
         padding-left: 80px; /* preenchimento esquerda com 80px */
      }

Proriedade abreviada padding

Para encurtar o código, é possível especificar todas as propriedades padding em uma propriedade abreviada padding.

A propriedade padding é uma propriedade abreviada para as seguintes propriedades de preenchimento individuais:

      Exemplo de preenchimento individual.
      padding-top     // preenchimento superior
      padding-right   // preenchimento direita
      padding-bottom  // preenchimento inferior
      padding-left    // preenchimento esquerda

Se a propriedade padding tiver quatro valores:

  • preenchimento: 25px 50px 75px 100px;
    • preenchimento superior é 25px
    • preenchimento direita é 50px
    • preenchimento inferior é 75px
    • preenchimento esquerda é 100px

Use a propriedade abreviada padding com quatro valores:

      Exemplo
      p {
        /* propriedade abreviada */ 
        padding: 25px 50px 75px 100px;
      }

Se a propriedade padding tiver três valores:

  • padding: 25px 50px 75px;
    • preenchimento superior é 25px
    • os preenchimento direita e esquerda são 50px
    • e o preenchimento inferior é 75px

Use a propriedade abreviada preenchimento com três valores: 

      p {
        // propriedade abreviada com três valores
        padding: 25px 50px 75px;
      }

Se a propriedade padding tiver dois valores:

  • padding: 25px 50px;
    • os preenchimento superior e inferior são 25px
    • os preenchimento direita e esquerdo são 50px

Use a propriedade abreviada de preenchimento com dois valores:

      propriedade abreviada com dois valores
      p {
        padding: 25px 50px;
      }

Se a propriedade padding tiver um valor:

  • padding: 25px;
    • todas os quatro preenchimentos são 25px

Use a propriedade abreviada preenchimento com um valor:

      Propriedade abreviada com um valor.
      p {
        padding: 25px;
      }

Preenchimento e largura do elemento

A propriedade CSS width especifica a largura da área de conteúdo do elemento. A área de conteúdo é a parte dentro do preenchimento, borda e margem de um elemento ( o modelo de caixa ).

Portanto, se um elemento tiver uma largura especificada, o preenchimento adicionado a esse elemento será adicionado à largura total do elemento. Isso geralmente é um resultado indesejável.

Exemplo

Aqui, o elemento <div> recebe uma largura de 300px. No entanto, a largura real do elemento <div> será 350px (300px + 25px de preenchimento esquerdo + 25px de preenchimento direito):

Padding e o elemento width

   
      <style>
      div.ex1 {
        width: 300px;
        padding: 10px;
        background-color: yellow;
        color: black;
        text-align: center;
        margin: auto;
      }
      
      div.ex2 {
        width: 300px;
        padding: 25px;
        background-color: lightblue;
        color: black;
        margin: auto;
      }
      </style>
Este div tem 300px de largura.

A largura deste div é de 350px, embora seja definido como 300px no CSS.
