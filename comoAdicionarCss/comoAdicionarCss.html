Como adicionar CSS

Quando um navegador lê uma folha de estilo, ele formata o documento HTML de acordo com as informações da folha de estilo.

Maneiras de inserir CSS

Existem três maneiras de inserir uma folha de estilo:

  • CSS externo
  • CSS interno
  • CSS embutido (não deve ser usado)

CSS externo

Com uma folha de estilo externa, você pode alterar a aparência de um site inteiro alterando apenas um arquivo!

Cada página HTML deve incluir uma referência ao arquivo de folha de estilo externo dentro do elemento <link>, dentro da seção <head> < link > </head>.

Exemplo

Estilos externos são definidos dentro do elemento , dentro da seção de uma página HTML:

   <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, 
      initial-scale=1.0">
      <title>Como Adicionar CSS<title>
   
      <!-- EXEMPLO DE CSS EXTERNO -->
      <!-- 
         href => mostra o endereço do arquivo 
                 onde se encontra a CSS
      -->
      <link rel="stylesheet" href="mystyle.css">   
   </head>

Uma folha de estilo externa pode ser escrita em qualquer editor de texto e deve ser salva com uma extensão nomeDoArquivo.css.

O arquivo .css externo não deve conter tags HTML.

Aqui está a aparência do arquivo "mystyle.css":

"mystyle.css"

   p#externo { /* p é o seletor */
      
      /* cor do fundo azul claro */
      background-color: lightblue;
      
      /* color é a propriedade
         navy é o valor */
      color: navy;
      
      /* margin-left é a propriedade
         20px é o valor */ 
      margin-left: 20px;
   }
   /* cor das letras marinho e a 
      margem esquerda com 20 pixel */

Resultado do Exemplo acima

Este parágrafo será estilizado pelo CSS externo.

Nota: Não adicione um espaço entre o valor da propriedade e a unidade:
Incorreto (com espaço): margin-left: 20 px;
Correto (sem expaço): margin-left: 20px;

CSS interno

Uma folha de estilos interna pode ser usada se uma única página HTML tiver um estilo exclusivo.

Os estilos internos são definidos dentro do elemento <style>, dentro da seção <head> de uma página HTML:

   <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, 
      initial-scale=1.0">
      <title>Como Adicionar CSS<title>
      
      <!-- EXEMPLO DE CSS INTERNO -->
      <style>
         p#interno { /* p é o seletor */
            
            /* cor do fundo azul claro */
            background-color: lightblue;
            
            /* color é a propriedade
               navy é o valor */
            color: navy;
            
            /* margin-left é a propriedade
               20px é o valor */ 
            margin-left: 20px;
         }
         /* cor das letras marinho e a 
            margem esquerda com 20 pixel */
      </style>   
   </head>

Resultado do Exemplo acima

Este parágrafo será estilizado pelo CSS interno.

CSS embutido

Um estilo embutido pode ser usado para aplicar um estilo único a um único elemento.

Para usar estilos embutidos, adicione o atributo style ao elemento relevante. O atributo style pode conter qualquer propriedade CSS.

Exemplo CSS Embutido

Os estilos embutidos são definidos no atributo style do elemento relevante (CSS embutido não devem ser mais utlizados):

   NÃO SE USA MAIS CSS EMBUTIDO NAS TAGs HTML

   /* h2 com fundo preto, letras vermelhas, centralizado */
   <h2 style="background-color: black; color: red; 
   text-align: center;">Este é um título</h2>

   /* p com fundo verde, letras amarelas, à esquerda. */
   <p style="background-color:green; color: yellow">Este parágrafo       
   será estilizado pelo CSS embutido.</p>

Resultado do Exemplo Acima

NÃO SE USA MAIS CSS EMBUTIDO NAS TAGs HTML

Este é um título

Este parágrafo será estilizado pelo CSS embutido.

OBS: Estilos CSS embutidos não devem ser usados, mova estilos para um arquivo CSS externo.

Comentários CSS

Os comentários CSS não são exibidos no navegador, mas podem ajudar a documentar seu código-fonte.

Os comentários são usados ​​para explicar o código e podem ajudar quando editar o código-fonte posteriormente.

Os comentários são ignorados pelos navegadores.

Os comentários CSS começa com barra / asterisco * ( / * ) e termina com asterisco * barra / ( * / ) Tudo que estiver dentro deles é um comentário.

   /* eu sou um comentário de uma linha */

      /* eu sou 
         um comentário
         de várias
         linhas
      */

      /* <p>Este parágrafo vai virar um comentário. </p> */
